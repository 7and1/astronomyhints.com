# Texture Assets

This directory should contain planet texture images. You can download high-quality textures from:

## Recommended Sources:

1. **Solar System Scope** (https://www.solarsystemscope.com/textures/)
   - Free 2K textures for all planets
   - Download and rename as follows:
     - mercury.jpg
     - venus.jpg
     - earth.jpg
     - mars.jpg
     - jupiter.jpg
     - saturn.jpg
     - uranus.jpg
     - neptune.jpg

2. **NASA's Visible Earth** (https://visibleearth.nasa.gov/)
   - High-resolution Earth textures

3. **Planet Pixel Emporium** (http://planetpixelemporium.com/)
   - Professional quality planet maps

## Fallback:
If textures are not available, the planets will render with solid colors defined in the store.
